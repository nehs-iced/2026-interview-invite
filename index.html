🛠️ 開發組（Python）｜打造專屬網站與 Discord 機器人

想要親手打造網站？或讓你的 Discord Server 擁有專屬機器人？從零開始掌握 Python Flask 與 Discord Bot 開發，完成可上線的實作專案！

  • 社課地點：科藝大樓二樓 國中資訊教室
